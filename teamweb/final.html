Final Solution

ArcGIS FieldMaps

We chose ArcGIS Field Maps because it is a native application that can consume offline packages. It is Esri's solution for offline mapping allowing you to access many operational layers with minimal setup. It is intuitive to use with a good UI and requires no particular knowledge of GIS to operate.

    Pros:
  1. Offline Capability
  2. Easy to configure offline areas
  3. Requires no coding experience
  4. Data sync is easy betweem offline map and online map
  5. Allows multi user edits with little conflict
Cons:
  1. Requires a native app to view results/solutions
  2. Depending on the area, offline package file size could be really large.

ArcGIS Field maps is the only solution explored that met our ideal. An application that consumes a map service layer to be used offline in case of emergencies. QField came close and may be a contender but we were unable to load our test data into it. The other web based solutions could not functionaly veiw the map layers without frequent server requests.
