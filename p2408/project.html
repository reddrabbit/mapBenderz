Project Overview

Research, explore, test, and compare as many Web based applications with offline map functionality within our given time limit based on a set of criteria.

Clients

Research, explore, test, and compare as many Web based applications with offline map functionality within our given time limit based on a set of criteria.

Objectives

In emergency situations maps are still needed, even if internet connection is lost. if the map is not accessible offline, in the event of internet outage, personel must rely on paper maps.

Technology

Emergency response teams on the field would have a hard time carrying out their tasks if there is no network connectivity for them to access web maps over the net.
