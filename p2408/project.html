Project Overview

Toolkit for supporting municipal wide emergency response; includes an offline dynamic map, web application and hazard analysis to identify at risk areas.

  1. An Offline Mobile Map Package with update script.
  2. An updated Situational Awareness Web Application
  3. Risk Assessment
  4. Network Analysis
  5. Dynamic Basemap

Clients

Municipality of Clarington

Objectives

  1. Enable access to private municipal Geospatial data when internet connectivity is unavailable or unstable.
  2. The created product can be regularly updated to ensure the data is contains the latest information when it is required
  3. The created product should support and meet the client's needs with appropriate functions and features
  4. Support municipal emergency response by identifying most at-risk areas.
