A truely offline dynamic Map was achieved with the esri file format mmpk (mobile map package). This file is created in ArcGIS pro and veiwed in esri Fieldmaps. Advantages of Using an MMPK file

  • Entire Study Area fits in one map
  • NO esri License required to view
  • Offline Location searching of Addresses, Businesses and Community Services
  • Dynamic Labeling
  • Custom interactivity
  • Plus built in Esri Fieldmaps tools (measure, Markup, ect.)

Update Script

The mmpk file can not be editted once created, but in our changing world, new roads are built, businesses close, and lots are severed, regularily. To keep this emergency response map upto date it must be regenerated regularly. The best way to do this is with a scedualed Arcpy script. Once the symbolization and popups are initially set up and saved, a Arcpy script can create a new file.

This script:

  1. Pulls the specified data layers from the server
  2. Recreates Business, Address, and Community Service locators with current data
  3. Packages the map into a .mmpk file format
  4. Sends an Email notification of completion or error report
